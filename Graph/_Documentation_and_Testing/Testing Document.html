Testing Document

 

This program takes a single Netdraw .VNA file as an command line argument.  It parses the information from the file line by line and creates a graph with the collected vertex and arc data specified by the .VNA file.  The program then collects and outputs basic statistics on the graph.  Finally it follows the depth-first search algorithm to find the Strongly Connected Components of the graph and outputs the Strongly Connected Components and its members in non-increasing order.

 

Sample .vna files included

 

SCC-Test.vna

celegansneural.vna
